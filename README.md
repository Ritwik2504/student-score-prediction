# 🎯 Student Score Prediction using Machine Learning

## 📌 Project Overview

This project aims to predict student exam scores based on study-related factors such as hours studied, attendance, and sleep hours. It demonstrates a complete machine learning workflow from data analysis to model evaluation.

---

## 🧠 Models Used

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor

---

## 📊 Evaluation Metric

* Mean Absolute Error (MAE)

---

## 📈 Key Insights

* Random Forest achieved the lowest MAE, making it the best-performing model
* Hours studied was the most influential feature
* Model predictions closely matched actual exam scores

---

## 📷 Visualizations Included

* Histogram of exam scores
* Actual vs Predicted values plot
* Residual vs Prediction plot
* Feature Importance graph
